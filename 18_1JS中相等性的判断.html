三种不同的值比较
  • 严格相等 ===
    • NaN === NaN --> false
    • -0 === +0 -->true
  • 抽象相等 ==
    • 比较时， 会发生隐式类型转换
    • NaN == NaN --> false
    • -0 == false --> true
    • +0 == false --> true
  • ES6新特性 Object.is
    • 与三等号相同（不进行类型转换），但是对于NaN和-0 和+0进行了特殊处理
      • Object.is(NaN, NaN) -->true
      • Object.is(-0,+0) --> false
      • Object.is(0,-0) --> false
